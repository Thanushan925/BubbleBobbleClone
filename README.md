# BubbleBobbleClone
Refactoring a PyGame Zero Bubble Bobble Clone

---

## How to Run the Game

### 1. Create and activate a virtual environment
```bash
python3 -m venv venv
source venv/bin/activate   # macOS
```

### 2. Install Dependencies
`pip install pgzero pygame`

### 3. Run game
`pgzrun main.py`
